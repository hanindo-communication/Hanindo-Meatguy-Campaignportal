# Meatguy Steakhouse — Landing Page

One-pager untuk mengarahkan traffic dari iklan video ke **WhatsApp** (reservasi / inquiry). Premium, immersive, satu CTA jelas.

## Isi project

- **`index.html`** — Halaman utama (buka di browser atau deploy sebagai static).
- **`content.json`** — Semua teks, link, dan copy one-pager dalam JSON; halaman memuat konten dari sini.
- **`PROJECT_SPEC.md`** — Brief dan syarat project.

## Cara pakai

1. **Lokal:** Buka `index.html` di browser (double-click atau drag ke browser). Atau jalankan `npx serve` di folder ini lalu buka http://localhost:3000 (agar `content.json` ter-load).
2. **Deploy:** Upload folder ini ke Vercel, Netlify, atau static host; set root ke folder ini dan file default ke `index.html`. Pastikan `content.json` ikut di-deploy.

## Ganti workspace ke project ini

Di Cursor: **File → Open Folder** → pilih folder `meatguy-landing`. Workspace aktif jadi Meatguy Steakhouse.
